Features

Solution Hunter has advanced Genetic Algorithms methods to allow this type of search engine to be even faster and more effective.

  • Using Hunter requires that it "understand" the type of problem as well as having a list of the things that influence it. It generally divides problems into "static" or "pattern" types.
  • Hunter provides a number of "plug-ins" targeted for different problem areas.
  • Hunter can be used by large or small businesses, or individuals.
  • It can run on a variety of systems Azure, AWS, or corporate Systems and is being configured for packaging in "containers" allowing it to run on Windows, Linux, and others.
  • It can run on single computers or multiple local or remote systems for additional speed.
  • Hunter is highly configurable and can learn to improve its strategy to solve the current problem more efficiently. When it finds a better “hunter”, it will dynamically switch to the more powerful image without interruption.
